# 湖南科技大学本科毕业论文格式 Skill

## 简介

此skill用于帮助检查和修复湖南科技大学本科毕业论文（设计）的格式，使其符合《湖南科技大学本科生毕业设计（论文）要求与撰写规范》。

**核心功能：**
- 全面检查docx文档格式是否符合规范
- 自动修复字体、字号、段落、引用标注、页眉页脚等格式问题
- 插入分节符并设置正确的页码格式（目录罗马数字 + 正文阿拉伯数字）
- 从零生成符合规范的示例docx文档
- 逐项验证修复后的文档格式

## 使用流程

典型使用顺序：

```
1. 检查格式 → python check_thesis.py 论文.docx
2. 修复格式 → python fix_thesis.py 论文.docx
3. 修复页码 → python fix_pagenum.py 论文.docx
4. 验证结果 → python verify_thesis.py 论文.docx
```

或从零生成规范文档：

```
python generate_thesis.py 格式示例.docx
```

## 脚本说明

### 1. check_thesis.py — 格式检查

检查文档格式是否符合规范，输出结构化错误报告。

**覆盖的检查项：**
- 页面设置（A4、页边距、页眉页脚距离）
- 页眉（内容、字体、字号、居中，从目录页开始）
- 页码（罗马/阿拉伯数字格式、字体、居中）
- 字体字号（章/节/条/款/正文各级标题和内容）
- 段落格式（行距1.25倍、首行缩进2字符、段前段后间距）
- 摘要格式（中英文标题、关键词标签、摘要内容字体）
- 关键词数量（3~5个）
- 引用标注（上标格式、不得在标题处）
- 公式格式（居中）
- 表格格式（表序按章、表名居中无标点、表内文字字体字号）
- 插图格式（图序按章、图题在图下）
- 每章另起页（含摘要、目录、参考文献、致谢、附录）
- 章编号格式（大写数字，如"第三章"而非"第3章"）
- 标题规范（章标题≤15字、无标点符号、论文题目≤20字）
- 致谢格式（标题居中四号加粗）
- 附录格式（标题居中，字号与章标题同级）
- 参考文献条目中英文混排字体
- 年份格式（四位数全写）
- 内容结构完整性

**用法：** `python check_thesis.py 论文.docx`

### 2. fix_thesis.py — 格式修复

自动修复文档格式问题。

**修复项目：**
- 页面设置（A4、页边距、页眉页脚距离）
- 页眉（五号宋体居中）
- 字体字号（按章/节/条/款/正文级别）
- 段落格式（行距、首行缩进、段前段后间距）
- 摘要格式（中英文标题、关键词标签）
- 致谢格式（标题四号宋体加粗居中）
- 附录格式（标题小二号宋体加粗居中）
- 引用标注（设为上标格式）
- 参考文献格式（标题居中、条目顶格）
- 表内文字格式（中文五号宋体、英文五号TNR）

**用法：** `python fix_thesis.py 论文.docx [输出文件.docx]`

### 3. fix_pagenum.py — 页码修复

在文档中插入分节符，设置正确的页码格式。

**功能：**
- 检测文档是否有目录
- 在第一章前插入分节符
- 节1（目录等前置部分）：罗马数字页码 `-i-`, `-ii-`...
- 节2（正文部分）：阿拉伯数字页码 `-1-`, `-2-`...
- 页脚设置为 `-X-` 格式，五号Times New Roman居中

**用法：** `python fix_pagenum.py 论文.docx`

### 4. generate_thesis.py — 生成示例文档

从零生成符合规范的示例docx文档（基于规范附件1-10的示例内容）。

**生成内容：**
- 扉页（附件1）
- 任务书（附件2）
- 指导人评语（附件3）
- 评阅人评语（附件4）
- 答辩记录（附件5）
- 中英文摘要+关键词（附件6）
- 目录（附件7）
- 正文：第一章前言 + 第二章发展阶段 + 第五章结论（附件8）
- 表格示例：表2.1 外汇储备情况（附件8+11）
- 插图示例：图3.1 催化剂XRD图谱（附件11）
- 参考文献9条（附件9）
- 致谢（附件10）
- 附录A：主要源程序清单

**用法：** `python generate_thesis.py [输出路径]`

### 5. verify_thesis.py — 格式验证

全面验证修复后的文档格式，逐项对照规范检查。

**用法：** `python verify_thesis.py 论文.docx`

## 规范速查表

| 项目 | 要求 |
|------|------|
| 纸张 | A4 (210mm×297mm) |
| 上页边距 | 30mm |
| 下页边距 | 25mm |
| 左页边距 | 25mm |
| 右页边距 | 25mm |
| 页眉距离 | 20mm |
| 页脚距离 | 15mm |
| 页眉内容 | 湖南科技大学本科生毕业设计（论文） |
| 页眉字体 | 五号宋体(10.5pt)居中 |
| 页码格式 | 前置部分 -i- 罗马数字，正文 -1- 阿拉伯数字 |
| 页码字体 | 五号Times New Roman(10.5pt)居中 |
| 章标题 | 宋体加粗小二号(18pt)居中，段前段后各0.5行(6pt) |
| 节标题 | 宋体加粗四号(14pt)顶格，段前段后各0.5行 |
| 条标题 | 宋体加粗小四号(12pt)顶格，段前段后各0.5行 |
| 款/项标题 | 宋体加粗小四号(12pt)，段前段后不空，1.25倍行距，首行缩进2字符 |
| 正文 | 宋体小四号(12pt)，段前段后不空，1.25倍行距，首行缩进2字符 |
| 中文摘要标题 | 小二号宋体加粗居中 |
| 英文摘要标题 | 小二号Times New Roman加粗居中，另起页 |
| 关键词标签 | 中文：黑体小四号；英文：Times New Roman加粗小四号 |
| 关键词数量 | 3~5个 |
| 目录标题 | 黑体三号字(16pt) |
| 目录条目 | 章标题条目宋体加粗四号(14pt)，节/条条目宋体小四号(12pt) |
| 参考文献标题 | 四号宋体加粗居中 |
| 参考文献条目 | 顶格，五号宋体(10.5pt)，英文部分用Times New Roman |
| 致谢标题 | 宋体四号加粗(14pt)居中 |
| 附录标题 | 宋体加粗小二号(18pt)居中，编排与正文相同 |
| 引用标注 | 上标，方括号阿拉伯数字；直接说明时TNR与正文排齐 |
| 表格 | 三线表，表序按章(表X.Y)，表名居中无标点；表内中文五号宋体/英文五号TNR |
| 插图 | 图序按章(图X.Y)，图题在图下；分图用a)/b)；插图与图题不得跨页 |
| 标题要求 | 章标题≤15字，无标点符号，无英文缩写；论文题目≤20字 |
| 章编号 | 使用大写数字（如"第三章"而非"第3章"） |
| 每章另起页 | 章标题前应有分页符；摘要、目录、参考文献、致谢、附录也需另起页 |
| 年份格式 | 四位数全写（如"2008年"不能写成"08年"） |

## 常见问题

1. **页码格式不对**：先运行 `fix_pagenum.py` 插入分节符，再运行 `fix_thesis.py` 修复页脚字体
2. **引用标注不是上标**：运行 `fix_thesis.py` 自动将 [1],[2,3] 等设为上标
3. **字体显示不正确**：`fix_thesis.py` 同时设置中文字体和东亚字体，确保在Word中正确显示
4. **文件被锁定**：关闭Word后再运行脚本，或保存到不同文件名
