# 湖南科技大学论文格式脚本说明

## 脚本列表

| 脚本 | 功能 | 输入 | 输出 |
|------|------|------|------|
| `check_thesis.py` | 全面格式检查 | docx文件 | 错误/警告/通过列表 |
| `fix_thesis.py` | 格式修复 | docx文件 | 修复后的docx文件 |
| `fix_pagenum.py` | 页码分节修复 | docx文件 | 带页码的docx文件 |
| `generate_thesis.py` | 生成示例文档 | 无/输出路径 | 规范示例docx |
| `verify_thesis.py` | 格式验证 | docx文件 | 通过/失败报告 |

## 依赖

- Python 3.7+
- python-docx
- lxml

安装：`pip install python-docx lxml`

## 使用示例

### 检查现有文档
```bash
cd hnust-thesis/scripts
python check_thesis.py "D:\论文.docx"
```

### 修复格式
```bash
python fix_thesis.py "D:\论文.docx" "D:\论文_fixed.docx"
```

### 修复页码
```bash
python fix_pagenum.py "D:\论文_fixed.docx"
```

### 验证修复结果
```bash
python verify_thesis.py "D:\论文_fixed.docx"
```

### 生成示例文档
```bash
python generate_thesis.py "D:\格式示例.docx"
```

## 检查项覆盖范围

所有脚本覆盖《湖南科技大学本科生毕业设计（论文）要求与撰写规范》中以下章节：

- 3.2 章节及层次（标题长度、标点符号、缩写词）
- 3.3 引用文献（上标格式、位置）
- 3.6 外文字母正斜体
- 3.7 数字（年份格式）
- 4.1 页面要求（A4、页边距、页眉、页码）
- 4.2 字体字号段落（章/节/条/款/正文）
- 4.3 正文层次（每章另起页、孤行控制）
- 4.4 公式编排
- 4.5 插表编排
- 4.6 插图编排
- 4.7 参考文献
- 4.8 封面及扉页
- 4.9 摘要及关键词
